Chapter 2: Operations with Complex Numbers

2.1 Basic Operations with Complex Numbers

Complex number operations are fundamental in AC circuit analysis, especially for calculating impedance, voltage drops, and power. We handle real and imaginary parts separately when performing operations on complex numbers in rectangular form.

Addition and Subtraction

Addition: For z1 = a + jb and z2 = c + jd,

z1 + z2 = (a + c) + j(b + d)

Subtraction:

z1 - z2 = (a - c) + j(b - d)

Example:
If z1 = 3 + 4j and z2 = 2 + j:

  • Sum: z1 + z2 = 5 + 5j
  • Difference: z1 - z2 = 1 + 3j

Multiplication in Rectangular Form

When multiplying complex numbers, apply the distributive property, keeping j2 = -1 in mind.

Formula: For z1 = a + jb and z2 = c + jd,

z1 ⋅ z2 = (a ⋅ c - b ⋅ d) + j(a ⋅ d + b ⋅ c)

Example:
For z1 = 3 + 2j and z2 = 1 + j, the product is:

z1 ⋅ z2 = 1 + 5j

2.2 Division with Complex Conjugates

Division of complex numbers is simplified by using the complex conjugate of the denominator. For z1 = a + jb and z2 = c + jd:

  1. Multiply the numerator and denominator by the conjugate z2* = c - jd:

    2. z1 / z2 = ((a + jb)(c - jd)) / ((c + jd)(c - jd))

  2. Simplify, using j2 = -1:

    3. z1 / z2 = ((ac + bd) + j(bc - ad)) / (c2 + d2)

Example:
For z1 = 3 + 2j and z2 = 1 + j, multiply by 1 - j, yielding 1.5 - 0.5j.

2.3 Operations in Polar Form

Polar form simplifies multiplication and division by handling the magnitude and phase angle directly.

  • Multiplication: Multiply magnitudes and add angles.

    • z1 ⋅ z2 = (A1 ⋅ A2) ∠ (θ1 + θ2)

  • Division: Divide magnitudes and subtract angles.

    • z1 / z2 = (A1 / A2) ∠ (θ1 - θ2)

Example:
If z1 = 4 ∠ 30° and z2 = 2 ∠ 15°:

  • Product: z1 ⋅ z2 = 8 ∠ 45°
  • Quotient: z1 / z2 = 2 ∠ 15°

2.4 Magnitude and Phase of Complex Numbers

The magnitude and phase angle describe the size and orientation of a complex number in AC circuits.

  • Magnitude: For z = a + jb,

    • |z| = √(a2 + b2)

  • Phase Angle (θ):

    • θ = tan-1 (b / a)

Example:
For z = 3 + 4j:

  • Magnitude: |z| = 5
  • Phase Angle: θ = 53.13°
